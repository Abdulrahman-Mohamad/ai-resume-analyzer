# AI Resume Analyzer

An AI-powered web app that analyzes your resume and matches it against job descriptions, giving you a compatibility score and actionable feedback.

## ✨ Features
- Upload your resume for instant AI analysis
- Paste a job title and responsibilities to get a match percentage
- See exactly how well your resume fits a specific job posting

## 🛠️ Tech Stack
- React, JavaScript
- Puter.js (AI integration)

## ⚠️ Note
Live demo is temporarily unavailable. Clone the repo and run locally to test.

## ⚙️ Getting Started
```bash
npm install
npm run dev
```
